Política de Privacidad de RayoGo

Esta Política de Privacidad describe cómo RayoGo recopila, usa y protege la información de los usuarios. RayoGo es una aplicación móvil de simulación de delivery desarrollada con fines académicos. Al usar la aplicación, el usuario acepta las prácticas descritas en esta política.

1. Información que puede recopilar la aplicación

Dependiendo de las funciones utilizadas, RayoGo puede recopilar o procesar los siguientes datos:

  • Nombre, correo electrónico y datos básicos de perfil.
  • Datos de autenticación, incluyendo inicio de sesión con Google OAuth o credenciales de acceso.
  • Rol del usuario dentro de la app, por ejemplo cliente, repartidor o administrador.
  • Direcciones de entrega registradas por el usuario.
  • Ubicación aproximada o precisa, cuando se usa para mapas, rutas, seguimiento o funciones de reparto.
  • Historial de pedidos, productos seleccionados, estado de pedido y datos de entrega.
  • Imágenes cargadas por el usuario, como comprobantes o evidencias usadas únicamente para la simulación del proceso de pago o entrega.
  • Datos técnicos necesarios para el funcionamiento de la app, como token de sesión, conexión con el servidor, errores o información básica del dispositivo.

2. Finalidad del uso de datos

Los datos se utilizan para:

  • Permitir el registro e inicio de sesión del usuario.
  • Gestionar pedidos, carrito, productos, direcciones y entregas.
  • Mostrar mapas, rutas y ubicación relacionada con el servicio simulado.
  • Asignar pedidos a repartidores y actualizar el estado de entrega.
  • Validar de forma simulada el método de pago seleccionado.
  • Mantener la seguridad de la sesión y el correcto funcionamiento de la aplicación.
  • Atender solicitudes de soporte, eliminación de cuenta o eliminación de datos.

3. Inicio de sesión con Google OAuth

Si el usuario elige iniciar sesión con Google, la aplicación puede recibir información básica de la cuenta, como nombre, correo electrónico e identificador de usuario. Esta información se usa únicamente para autenticar al usuario y crear o acceder a su cuenta dentro de RayoGo.

4. Uso de ubicación

RayoGo puede solicitar acceso a la ubicación del dispositivo para funciones de mapas, cálculo de rutas, seguimiento del pedido o gestión del reparto. La ubicación se usa únicamente para las funciones operativas de la aplicación y no se vende a terceros.

5. Imágenes y comprobantes

Si la aplicación permite cargar imágenes, estas se usan únicamente para fines de validación dentro de la simulación de delivery, por ejemplo comprobantes de pago o evidencias relacionadas con pedidos. RayoGo no procesa pagos reales ni almacena datos de tarjetas bancarias.

6. Servicios de terceros

Para funcionar, RayoGo puede utilizar servicios externos relacionados con autenticación, mapas, almacenamiento, base de datos, servidor backend y comunicación de red. Estos servicios pueden procesar datos únicamente cuando sea necesario para operar las funciones de la aplicación.

Entre los servicios que pueden intervenir se encuentran:

  • Google Sign-In / OAuth para autenticación.
  • Google Maps o servicios de ubicación para mapas y rutas.
  • Servidor backend y base de datos usados para registrar usuarios, pedidos y entregas.

7. Compartición de datos

RayoGo no vende datos personales. La información puede ser procesada por servicios necesarios para el funcionamiento de la app, como autenticación, mapas, servidor backend, almacenamiento o base de datos. También podría compartirse información si fuera requerido por una obligación legal aplicable.

8. Seguridad

Se aplican medidas razonables para proteger la información del usuario, como control de acceso, autenticación y comunicación con servicios del backend. Aun así, ningún sistema digital puede garantizar seguridad absoluta.

9. Conservación de datos

Los datos se conservan mientras sean necesarios para el funcionamiento de la app, para mantener la cuenta del usuario o para cumplir fines académicos y de soporte. El usuario puede solicitar la eliminación de su cuenta y datos asociados.

10. Eliminación de cuenta y datos

El usuario puede solicitar la eliminación de su cuenta y datos personales escribiendo a leonel06tipan@gmail.com o siguiendo las instrucciones de la página Eliminar cuenta y datos.

11. Privacidad de menores

RayoGo no está dirigida a niños ni recopila intencionalmente información de menores de edad. Si se detecta información de un menor, se puede solicitar su eliminación mediante el correo de contacto.

12. Cambios en esta política

Esta política puede actualizarse cuando existan cambios en la aplicación, en sus funciones o en los requisitos de Google Play. La versión actualizada se publicará en esta misma página.
